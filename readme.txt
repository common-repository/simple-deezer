This plugin add 2 shortcodes to easily put a Deezer Player into post content. (compatible 2.5, 2.6, 2.7)

== Description ==

*Usage*

Insert shortcode in your post content.

New Syntax : [dz id="123" alt="Alternative text"]
Old Syntax : [dz | 123 | alternative text]

== Installation ==

1. Upload `simple-deezer.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That all
